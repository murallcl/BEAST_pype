# Parameters

This directory houses yml files for parameterising BEAST_pype workflows. 
For notes on editing yml files see:
- https://www.datacamp.com/blog/what-is-yaml

The files starting with `Template_` are annotated templates that you can fill in.

The subdirectory  `sbatch_run_examples` contains some example yml files for running BEAST_pype workflows on a HPC via slurm's `sbatch` command.
The subdirectory  `locally_run_examples` contains some example yml files for running BEAST_pype workflows locally via command line.
